## Description
A modern and responsive Smart Home landing page built with HTML, SCSS, and JavaScript.

## Features

- Hide/Show the answer to a FAQ’s when the question is clicked.
- Display the hamburger menu icon on mobile devices.
- When the hamburger menu is clicked, it opens the mobile navigation menu.
